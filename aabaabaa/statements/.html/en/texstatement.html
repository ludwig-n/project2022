AABAABAA

Time limit 1 second
Memory limit 64.0 MB
Input stdin or input.txt
Output stdout or output.txt

You may have heard of the ABACABA sequence. The AABAABAA sequence is similar but each string in it contains three copies of the previous one, not two. Formally,

  • the 0th string in the sequence is an empty string,
  • for n ≥ 1 n \geq 1 n≥1 the n n n-th string in the sequence has the form S T S T S STSTS STSTS, where S S S is the ( n − 1 ) (n - 1) (n−1)-th string in the sequence and T T T is the uppercase n n n-th letter of the Latin alphabet.

Using recursion, generate the n n n-th string in the AABAABAA sequence.

Input format

An integer n n n is given ( 0 ≤ n ≤ 15 0 \leq n \leq 15 0≤n≤15).

Output format

Output the n n n-th string in the AABAABAA sequence.

Input Output 
0

AABAABAA
