Возможно, вы слышали о последовательности ABACABA. Последовательность AABAABAA похожа на неё, но в ней каждая следующая строка содержит три копии предыдущей, а не две. Формально,

  • 0-я строка в последовательности — пустая строка,
  • для n ≥ 1 n \geq 1 n≥1 n n n-я строка в последовательности имеет вид S T S T S STSTS STSTS, где S S S — ( n − 1 ) (n - 1) (n−1)-я строка в последовательности, а T T T — заглавная n n n-я (в нумерации с единицы) буква латинского алфавита.

С помощью рекурсии сгенерируйте n n n-ю строку в последовательности AABAABAA.

Формат ввода

Дано целое число n n n ( 0 ≤ n ≤ 15 0 \leq n \leq 15 0≤n≤15).

Формат вывода

Выведите n n n-ю строку в последовательности AABAABAA.
