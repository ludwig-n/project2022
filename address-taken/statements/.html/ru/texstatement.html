В Берляндию наконец-то провели интернет! Уже открылись два почтовых сервиса: Bmail и Bandex (изначально на них никто не зарегистрирован). Теперь все N N N жителей Берляндии хотят зарегистрировать себе по почтовому ящику. Каждый из них уже придумал себе желаемый почтовый адрес, но, к несчастью, некоторые из них могли совпасть. Все берляндцы по очереди пытаются зарегистрировать свои адреса, и каждый из них действует по следующему алгоритму:

  • сначала берляндец пробует зарегистрировать свой адрес на Bmail (так как этот сервис считается более престижным),
  • если на Bmail такой адрес уже зарегистрирован, то пробует зарегистрировать его на Bandex,
  • если он зарегистрирован и там, то сдаётся и остаётся без почтового ящика.

Для каждого берляндца определите, сможет ли он зарегистрировать адрес и на каком сервисе.

Формат ввода

В первой строке дано целое число N N N ( 1 ≤ N ≤ 1 0 5 1 \leq N \leq 10^5 1≤N≤105), затем даны N N N непустых строк, состоящих из строчных латинских букв и цифр — желаемые адреса берляндцев в том порядке, в каком они будут их регистрировать (каждый на отдельной строке). Суммарная длина всех адресов не превышает 1 0 5 10^5 105 символов.

Формат вывода

Для каждого из N N N берляндцев выведите на отдельной строке BMAIL или BANDEX, если он зарегистрируется на соответствующем сервисе, или FAIL, если не сможет это сделать ни на одном из них.

alice
