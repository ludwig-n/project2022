Birdwatching

Time limit 2.5 seconds
Memory limit 64.0 MB
Input stdin or input.txt
Output stdout or output.txt

Maya likes birdwatching. She identifies the species of every bird she sees and writes it down into a list. For each of these birds she wants to know if she has already seen a bird of the same species or from the same genus. But there are so many birds in her list that she can't do it by hand. Help her!

Input format

In the first line an integer N N N is given ( 1 ≤ N ≤ 1 0 5 1 \leq N \leq 10^5 1≤N≤105) — the number of entries in Maya's list. In each of the next N N N lines a scientific name of a bird species is given, consisting of two strings separated by a space. The first of these two strings is the name of the genus. For example, if Corvus cornix is the name of the species (hooded crow), then Corvus is the name of the genus it belongs to. Both strings consist of lowercase Latin letters, except for the first character of the genus name, which is an uppercase Latin letter. Both strings are 2 to 30 characters long (inclusive).

Output format

For each bird in Maya's list output

  • SPECIES if a bird of the same species has already appeared in the list,
  • GENUS if a bird in the same genus has already appeared in the list (but not a bird of the same species),
  • NEW if no birds in that genus have appeared in the list before.

Corvus cornix
Motacilla alba

Notes

Notice that in the last example the two birds are different species. The species is determined by both words in the name, not just the second one.
