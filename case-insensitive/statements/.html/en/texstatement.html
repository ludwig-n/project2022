Input stdin or input.txt

Determine if two strings are equal without taking the case of the letters into account (case-insensitively). For example, uppercase A and lowercase a should be considered the same letter.

Input format

Two strings 0 0 0 to 1 0 5 10^5 105 characters long (inclusive) are given, each on a separate line. The strings may contain uppercase and lowercase Latin letters, digits, spaces and ASCII punctuation (!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~).

Output format

Output YES if the two strings are equal without taking the case of the letters into account or NO otherwise.

equal
EQUAL

YES
