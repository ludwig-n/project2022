Ice Cream

Time limit 1 second
Memory limit 64.0 MB
Input stdin or input.txt
Output stdout or output.txt

Vasya wants some ice cream. Various ice cream flavors are sold in Vasya's town, and the same flavor can cost different prices in different shops.

Vasya has decided to try every ice cream flavor he sees. But he doesn't want to overpay. Determine the minimum total amount Vasya will have to spend to buy an ice cream of each flavor.

Input format

In the first line an integer N N N is given ( 1 ≤ N ≤ 1 0 5 1 \leq N \leq 10^5 1≤N≤105) — the total number of ice creams for sale in Vasya's town. In each of the next N N N lines a nonempty string and an integer are given (separated by a space) — the name of the ice cream flavor and the price of the ice cream in rubles. All flavor names consist of lowercase Latin letters. The total length of the flavor names does not exceed 1 0 5 10^5 105 characters. All prices are between 1 1 1 and 1 0 9 10^9 109 rubles.

Output format

Output the minimum total amount (in rubles) Vasya will have to spend to buy an ice cream of each flavor.
