In some messaging apps (such as Telegram, Discord and Slack) emoji can be written as a string (the name of the emoji) enclosed between colons. For example, :smile: turns into ðŸ˜„ and :christmas_tree: turns into ðŸŽ„.

You are working on a project analyzing emoji. You are given a text containing emoji written in this form. Your goal is to extract the names of the emoji from the text.

Input format

One string 0 0 0 to 1 0 5 10^5 105 characters long (inclusive) is given. The string may contain uppercase and lowercase Latin letters, digits, spaces and ASCII punctuation (!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~). The string contains several (possibly zero) emoji written in the form :emoji_name: where

  • emoji_name is a nonempty string that does not contain spaces or colons (the name of the emoji),
  • the first colon is always preceded by a space or the beginning of the string,
  • the second colon is always followed by a space or the end of the string.

It is guaranteed that each colon in the string is part of an emoji.

Output format

Output the names of the emoji in the order that they appear in the text, each on a separate line. Do not output colons. If the same emoji was used multiple times, you have to output its name multiple times.
